# 2025 BBS764 Course Module: Gene Regulatory Networks

This folder contains course materials for a module on Gene Regulatory Networks (GRN) analysis, including lectures, workshops, and homework assignments.

## Course Materials

### Lecture Materials

- `lecture.ipynb` - Lecture notebook with theory and examples
- `lecture.slides.html` - HTML slides for presentation (open in browser)

### Workshop Materials

- `workshop_student.ipynb` - Interactive workshop with exercises for students
- `workshop.ipynb` - Workshop with complete solutions (for instructors)
- `workshop_student.slides.html` - Workshop slides with placeholders
- `workshop.slides.html` - Workshop slides with solutions

### Homework Materials

- `homework.ipynb` - Homework assignments
- `homework_answers.ipynb` - Homework with complete answers (for instructors)
- `homework.slides.html` - Homework slides

## Using the Materials

### Viewing Slides

Open any `.slides.html` file in a web browser:

**Navigation tips:**
- Press `Space` or `→` to advance slides
- Press `←` to go back
- Press `Esc` to see slide overview
- Press `F` for fullscreen

### Running Notebooks

Download them to your own computer and open them with jupyter notebook. You should use jupyter notebook from the programming environment you already set up for earlier workshops of this course.

**For students**: Use `workshop_student.ipynb` which contains exercise placeholders.

**For instructors**: Use `workshop.ipynb` or `homework_answers.ipynb` which contain complete solutions.

### Course Content Overview

This module covers:
- Gene Regulatory Network fundamentals
- Differential expression analysis
- Network reconstruction and visualization
- Practical analysis using Python (pandas, matplotlib, networkx)
